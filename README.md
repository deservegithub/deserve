# บริษัท ดีเซิร์ฟ จำกัด — เว็บไซต์ (deseve.win)

เว็บไซต์หน้าแรก (Landing Page) ของบริษัท ดีเซิร์ฟ จำกัด
บริการ: รับทำเว็บไซต์ · รับเขียนโปรแกรม · รับทำแอปพลิเคชัน · รับทำโมบายแอป · การตลาดออนไลน์

## โครงสร้าง
- `index.html` — ไฟล์เว็บทั้งหมด (HTML + CSS + JS ในไฟล์เดียว, ไม่ต้อง build)

## การ deploy (Vercel)
เป็น static site — ไม่ต้องตั้งค่า build
- Framework Preset: **Other**
- Build Command: *(เว้นว่าง)*
- Output Directory: `./`

## โดเมน
ใช้งานที่ **https://deseve.win** (DNS ตั้งค่าที่ Cloudflare)
